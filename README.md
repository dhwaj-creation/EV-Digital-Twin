# EV-Digital-Twin

This project is a physics-based digital twin of an electric vehicle designed to simulate and optimize energy efficiency under real-world urban driving conditions.

## Motivation
Electric vehicles are often designed using standard driving cycles that do not fully represent real urban traffic. This project aims to explore how vehicle parameters affect efficiency using a physics-driven simulation approach.

## Current Status
 Project under development  
Phase 1: Vehicle physics modeling

## Planned Features
- Vehicle longitudinal dynamics model
- Energy consumption simulation
- Urban driving cycle analysis
- Parameter optimization (mass, regenerative braking, torque)
