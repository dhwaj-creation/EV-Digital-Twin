# EV Digital Twin

A Python-based EV Digital Twin simulator that models electric vehicle performance in real time.

## Features

- Battery simulation
- Range estimation
- Driving modes
- Energy consumption analysis
- Journey simulation
- Interactive Streamlit dashboard
- Graphs and visualizations

## Technologies Used

- Python
- Streamlit
- NumPy
- Pandas
- Matplotlib

## How to Run

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
